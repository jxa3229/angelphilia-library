## 4. 按 QUARANTOTTO 信件整理的上妆顺序

### Step 1：清洁

- 用中性洗剂或清水轻轻清洁 body 表面。
- 完全晾干。
- 不要在上色前先喷透明保护漆。

### Step 2：隐藏处试涂

- 选择不显眼位置，用准备正式使用的 Vカラー 或 Mr.カラー 试涂。
- 放置 24-72 小时。
- 确认没有发粘、咬底、开裂、变色后再继续。

### Step 3：大面积补色

- 如果优先考虑涂膜强度：用 Vカラー。
- 如果是大面积、需要更均匀的喷涂效果：用 Mr.カラー airbrush 薄喷。
- 薄喷多层，不要一次厚喷。

### Step 4：细小部位、阴影、腮红

- 使用 ゴンドラパステル 或 Holbein PanPastel。
- 用棉棒、海绵棒、软刷少量叠加。
- 适合细线、边缘、渐变、脸部柔和色。

### Step 5：确认整体颜色

- 检查整体色差。
- 如果不够，再少量补色。
- 不要急着封层，先确认颜色和质感。

### Step 6：最后消光保护

- 全部上色完成并完全干燥后，再喷 Mr.カラー 系消光喷罐。
- 推荐：Mr.スーパースムースクリアー つや消し B530。
- 薄喷 2-3 层，每层之间留出干燥时间。

## 5. 简化流程

清洁 -> 隐藏处试涂 -> Vカラー/Mr.カラー 补大面 -> Pastel 修细节 -> Mr.カラー 消光喷罐保护

## 6. 关键注意事项

- 上色前不喷保护漆；保护喷罐是最后一步。
- 上色前的“保护”只指 masking tape 遮蔽周边，不是透明 clear coat。
- Vカラー、Mr.カラー、稀释剂、喷罐都有溶剂风险，必须通风，建议佩戴有机溶剂防毒面具和丁腈手套。
- 喷罐和稀释剂跨境运输限制多，国内购买优先找“国内现货、陆运发货”的模型店。
- 任何材料正式上 body 前都要先在隐藏处测试。
- 工厂原厂涂料特殊，市售材料只能作为参考，无法保证同样完成效果。
